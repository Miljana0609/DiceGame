# Tärningsspel
Ett enkelt tärningsspel utvecklat som ett av mina första projekt. Projektet fokuserar på grundläggande objektorienterad programmering och spellogik.
Senare har ett enkelt frontend lagts till för att visualisera spelet.
# Teknik
Backend: JAVA ; Frontend: HTML/CSS/JavaScript ;
Grundläggande OOP
# Funktionalitet
- Två spelare
- Slår tärning och räknar poäng
- Visar resultat och vinnare
- Möjlighet att återställa spelet
